# Real-Time Poll Interaction App

This is a basic web application for creating, joining, and voting in live polls. It includes different pages for the admin and users.

# Features

- Admin can create a poll
- Users can join poll using session code
- Users can vote on the poll
- Live chart result is displayed
- Simple and clean UI

# How to Run

1. Download or clone the repository
2. Open the following HTML files in browser:
   - `poll.html` → Admin page
   - `join_poll.html` → User joins session
   - `vote.html` → Voting page
   - `result.html` → Live result chart

No server required – everything works with frontend files.

# Technologies Used

- HTML
- CSS
- JavaScript
- Chart.js (for result visualization)
